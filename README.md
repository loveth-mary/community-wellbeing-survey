## 🚀 Progress Update (Page 1 & Page 2)

Today, I worked on building and styling **Page 1 and Page 2** of my Community Well-being Survey project using HTML and CSS.

### 🧩 What I did
- Built the structure for Page 1 (Basic Information) and Page 2 (Living Conditions)
- Styled both pages to improve layout, spacing, and readability
- Added a subtle **box shadow** to give the UI a cleaner and more polished look

### 🐞 Issues I encountered
- Styling Page 2 affected my **consent page layout** (checkbox and text alignment broke)
- Input fields appeared wider than select dropdowns, causing inconsistency

### 🛠️ How I fixed them
- Scoped my styles by wrapping sections in a class (`.container`) to control where CSS applies
- Used `box-sizing: border-box;` to fix inconsistent input and select widths
- Adjusted layout to keep elements aligned properly across pages

### 💡 What I learned
- CSS can easily affect multiple pages if not properly scoped
- Small UI inconsistencies (like input sizes) can impact the overall look
- Structuring HTML properly makes styling much easier and cleaner
